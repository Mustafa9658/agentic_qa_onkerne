# Account Deletion on Example.com

## Goal: Permanently delete the user account by clicking the 'Delete My Account Forever' button on the website.

## Tasks:
- [ ] Navigate to https://example.com
- [ ] Log in to the user account if not already logged in
- [ ] Locate the 'Delete My Account Forever' button on the website
- [ ] Click the 'Delete My Account Forever' button
- [ ] Confirm the account deletion if a confirmation prompt appears
