# Greet User and Verify Homepage Loads Successfully

## Goal: Verify that the homepage loads and displays a greeting or welcome message to the user.

## Tasks:
- [ ] - [ ] Navigate to the website homepage URL
- [ ] - [ ] Verify the homepage loads without errors (page elements are visible, no 404 or server error)
- [ ] - [ ] Check for the presence of a greeting or welcome message (e.g., 'Hi', 'Hello', or similar) on the homepage
- [ ] - [ ] If the greeting is not visible, check for alternative welcome indicators (e.g., banners, pop-ups)
- [ ] - [ ] If the homepage fails to load or greeting is missing, document the error and capture a screenshot
